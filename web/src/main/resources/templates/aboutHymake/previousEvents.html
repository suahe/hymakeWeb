海迈科技历届办赛介绍

海迈科技作为省内建筑信息化龙头企业，不仅致力于将信息技术与建筑业的深度融合，推动产业要素集聚，促进数字经济发展。同时，也时刻关注着高校在建设领域人才的培养与发展，公司每年都会拨付出一部分资金，用于高校事业的支持发展，从2012年起，海迈科技联合省内高校，共举办省级高校技能竞赛七起，校级技能竞赛数十起，在办赛经验上有丰富的经验，大至数百人的同场竞赛，小至数十人的现场PK。从赛事的筹划、赛程的制定、赛事的规则、赛事的现场竞赛、赛后颁奖及宣传有着整套完整的流程计划与安排。通过多次办赛积累了许多专家、学者的资源，在竞赛考核环节，能做到高质量、高水平的评审，保证竞赛的公平，参赛作品的优质。

福建省建筑信息模型（BIM）大赛
/ 2018

2019年1月4日，由福建省住房和城乡建设厅指导，福建省建筑信息模型BIM应用联盟、福建省勘察设计协会、福建省工程建设科学技术标准化协会主办，厦门海迈科技股份有限公司等承办的福建省首届建筑信息模型（BIM）大赛在福州落下帷幕！

赛事以2013年福建省科技重大专项专题《建设行业信息一体化软件核心技术研发及应用》研究成果为大赛技术基础，通过现场、网络技能竞赛方式，为工程建设各方主体搭建切磋技术、交流经验、提高技能的平台，加快BIM应用人才的培养，进一步提高建筑职工现代技术水平，加快普及建筑信息模型（BIM）技术，积极推动全省建筑行业信息化技术发展。

第二届福建省高校建筑信息创新大赛
/ 2017

2017年11月10日，由福建省教育厅、福建省住房和城乡建设厅主办，厦门海迈科技股份有限公司（以下简称“海迈科技”）、福建江夏学院共同承办，福建省建筑信息模型技术应用联盟、腾讯网、福建教育电视台等协办的第二届福建省高校建筑信息创新大赛总决赛在福建江夏学院圆满落幕。大赛以“拥抱创新时代，建筑创业梦想”为主题，旨在深化高校创新创业教育改革，为建筑信息化技术应用创新选拔人才，以新一代信息和网络技术为支撑，加强技术应用、商业模式创新和在建筑设计、招标、施工和运维全过程的综合应用,推动建筑产业转型升级、提质增效和节能环保，促进平台经济、众包经济、分享经济等创新发展。赛事覆盖我省30所高校、近3000名师生报名参与，并于全省范围内展开校级选拔赛、全省总决赛，历时6个月的激烈角逐，优选BIM技术应用创新人才和商业模式，由600多个报名项目中评选出60个获奖项目进入现场决赛，实现了覆盖面更广、规模更大、影响力逐年提升的良好收效。

福建省高校建筑信息创新大赛
/ 2016

2016年10月30日，由省教育厅主办，厦门海迈科技股份有限公司、厦门理工学院、厦门软件职业技术学院共同承办的福建省高校建筑信息创新大赛在厦门落下帷幕。

大赛旨在探索高校与行业联合培养人才的新机制，深化“产学研用”融合，提高学生信息技术应用水平和创新创业能力，促进信息化技术应用创新人才的培养。大赛历时5个月，覆盖我省60多所院校，4万名建筑类、软件类等专业的学生，来自全省27所院校的53支队伍进入总决赛（本科院校队伍32支、高职院校队伍21支）的角逐。最终，大赛评出一等奖4个、二等奖6个、三等奖12个和优秀奖16个，同时还评出4个优秀创新创业导师奖、10个优秀组织奖、最佳创新奖和最具人气奖。

“海西杯”高校招投标大赛
/ 2015

2015年12月27日，由省教育厅主办，集美大学诚毅学院、厦门海迈科技股份有限公司共同承办的“海西杯”招投标大赛在集美大学诚毅学院落下帷幕。

/ 2013

/ 2012
